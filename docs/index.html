inventory quantity update (and base price!) on a single admin page

How to Install

  • for v157 installs, just copy the InvUpdator directory from the zc_plugins to your store zc_plugins directory. Go to Admin -> Modules -> Plugin Manager; select Inventory Updator and Install.
  • The Inventory menu option is now on the tools menu.
  • for v156 installs, copy the contents of the admin directory within the for_v156 to your YOUR_ADMIN directory. the directory structure should match.
  • If you are upgrading to the new zc_plugin v157 version, you need to delete the old versions of the files. You can manually do this; or within this file: zc_plugins/InvUpdator/1.2.3/Installer/ScriptedInstaller.php, on line 12 remove the 2 slashes, and the plugin installer should do it for you. See: 
    $this->removePrepluginVersion();
  • If you need help with FTPing the files up to your server, or an FTP client, go to the zen-cart forum.

    • Notes

  • There are NO core zen-cart files modified.
  • Now makes use of the zc v157 plugin installer! What could be easier?
  • With the plugin installer on v157, the inventory page is now on the tools menu.
  • There are no additional fields added to any zen-cart tables
  • The extra file for v156 installs is NOT the same as the v157 available on github. There is a slight difference due to changes between the 2 versions of ZC.
  • Update multiple items inventory quantity and base price fields within a category from a single page.
  • One can change the items status by clicking on the status icon; green go red (inactive) and vice versa.
  • Clicking on the item name will bring you directly to the item maintenance page.
  • No TESTING has been done with any other plug-ins; ie stock by attributes. If there is a demand, perhaps i will look at incorporating.
  • The category selection should match your category order in the category products listing.
  • When selecting a category that has sub-categories, you should see all products from all sub-categories; ie down 1 level.
  • If you have sub-sub-categories, you will need to drill down further.
  • Again, you will only see products down 1 level; to go down further, select a further sub-category.
  • When at the top of the category list you will see all products, irrespective of the number of category levels you have.
  • If there is a demand perhaps i will add the ability to search and then you can update inventory quantities of the items from your search results.
  • This new page is available on the TOOLS menu or at www.yoursite.com/YOUR_ADMIN/index.php?cmd=inventory
  • On v156, to add this module as an admin menu item, go to your Admin -> Admins-> Admin Page Registration.
    • Page Key -> adminInventory
    • Page Name -> BOX_ADMIN_INVENTORY
    • Page Filename -> FILENAME_INVENTORY
    • Page Parameters -> (leave blank)
    • Menu -> Your Choice!
    • Display on Menu -> check the box
    • Sort Order -> Pick a number!
